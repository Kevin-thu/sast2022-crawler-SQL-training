# GPA calculator

Calculate the GPA in each term automatically.

## Usage

1. Put your userid and password into `setting.json` in the same directory as `webvpn.py`. The format is like:
  ```json
  {
      "username": "20********",
      "password": "**********"
  }
  ```
2. Just run the code: `python3 webvpn.py`
